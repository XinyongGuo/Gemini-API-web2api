# Introduction
**Function**: This tool converts Gemini's web interface into a local API compatible with the OpenAI format, enabling integration with other applications.

**Usage**:
1. Download and extract the main.rar archive.
2. Execute setup.bat to automatically create a virtual environment.
3. Run run.bat to start the server, which will automatically execute a test script after 25 seconds.

**Note**: Ensure you are logged into Gemini in your browser.
